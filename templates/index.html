Welcome to the Movie Screening Service.

This is my final project in course "Python Web Service from Idea to Production by Tero Karvinen.

The meaning of this project is to provide a platform for indie movie workers and customers; a tool for both arranging movie screenings and attending to them.

The functionality of this website is simple: the host users create posts of movie screenings, with their own contact information attached to them.

The regular users, who would like to attend to the film screenings, can contact the hosts via email or phone -- registration is not necessary.


WARNING: THIS PROJECT IS STILL UNDER DEVELOPMENT -- PLEASE DO NOT USE THIS SERVICE IN PROFESSIONAL PURPOSES
